About Us

Overview

Pjuu is an open-source social networking application released under a "free as in freedom" license, the GNU AGPLv3.

We love choice and Pjuu is another option in the social networking sphere. It may not be as feature heavy as other social networking websites so if you want something lightweight it's a good option.

Being open source you can take the Pjuu code and run it on your own server. Anyone who is sensitive about the data they post on social networks can host it themseleves. These new servers can define there own terms of service and privacy policy and look after the data themselves. In the future we plan to allow these self-hosted Pjuu servers to communicate with one another.

If you are interested in seeing how Pjuu is build (its rather simple), check out our Github page.

Future Plans

There is still a lot of stuff we want to implement inside Pjuu. The following detail what we want to do before version 1.0:

  • Images We are planning image uploads in to the system for version 0.6. We could implement it now but we want a very scalable method of doing so. This will allow people who want to run Pjuu to follow a simple guide and have massively scalable file storage. This will also replace Gravatar, the current avatar images.
  • Privacy There is one problem with Pjuu as it stands, you can not control who can see your profile. To see a Pjuu profile you need to be signed up to see anything from any user. This will change in the future and you will be able to choose who can see your data, follow you and tag you in posts.
  • Groups It would be nice to have a group feature like those found on other social networks. This is still in the earlier thought process as to keep it super light and easy to use.
  • Private Messages Although private messaging is not a very social thing to do it is a must. We will be adding this in before version 1.0
  • Import/Export At the momment Pjuu's data dump system is very simple and is primarily so you can see what data you have posted to us. This will become better in future versions and we will enable to ability to export all your data and move it to another instance of Pjuu.

There are a few more things we would like to do but not before Pjuu is stable:

  • Federation This is planned in a bit further in the future. This will allow different people running there own copies of Pjuu to communicate. We beleive this is the way social networking will work in the future, it will stop a few big players from controlling your data selling you adverts.

If you have any ideas or something you would like to see inside Pjuu, open an issue over at Github.
