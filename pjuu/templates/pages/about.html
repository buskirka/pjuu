About Us

Overview

Pjuu is an open-source social networking application released under a "free as in freedom" license, the GNU AGPLv3.

We all love choice and Pjuu is another option in the social networking world. It may not be as feature-heavy as other social networking websites, so if you want something simple it's a good option.

As Pjuu is open source, you can take the Pjuu code and run it on your own server. Anyone who is sensitive about the data they post on social networks can host it themseleves. These new servers can define their own terms of service and privacy policy, and look after the data themselves. In the future we plan to allow these self-hosted Pjuu servers to communicate with one another.

If you are interested in seeing how Pjuu is build (its rather simple), check out our Github page.

Future Plans

There is still a lot of features we want to implement into Pjuu. The following details what we want to do before version 1.0:

  • Permissions Currently all users of the site have the ability to view your profile. To see a Pjuu profile you need to be signed up to see anything from any user. This will change in the future and you will be able to choose who can see your data, follow you, and tag you in posts.
  • Groups It would be nice to have a group feature like those found on other social networks. This is still in the early stages of the process to keep it super simple and easy to use.
  • Private Messages Although private messaging is not a very social thing to do, it is a must. We will be adding this in before version 1.0

If you have any ideas or something you would like to see inside Pjuu, open an issue over at Github, or create it yourself and send us a pull request.

Technology

Pjuu is a pretty simple software project, we have made some good decisions and some not so good, which we will have fixed before version 1.0. Below is a list of everything we use within Pjuu.

Services

  • Linode
    Our awesome web hosting company, the entire Pjuu website runs on top of Linode.
    https://www.linode.com
  • DigitalOcean
    Our backup and monitoring solution is hosted on a tiny node living on DO. This allows us to separate the components on Pjuu.
    https://www.digitalocean.com
  • Sentry
    Monitors the Pjuu application. If there are any issues we know straight away.
    https://getsentry.com

Software

  • Python
    A great open-source programming language.
    https://www.python.org
  • Debian
    All our guests run Linux and we love the Debian project.
    https://www.debian.org
  • Ansible
    Server orchestration software. Makes deploying servers a breeze.
    http://www.ansible.com
  • Nginx
    Awesome, rock solid, easy to use web server.
    http://nginx.org
  • Gunicorn
    Brilliant WSGI server for Python.
    http://gunicorn.org
  • Redis
    Formerly Pjuu's sole data store. Redis is amazing! Super fast and super flexible.
    http://redis.io
  • MongoDB
    A great document-based database. Allows easy scalability with sharding.
    http://www.mongodb.com

We may use more in the future but fot the time being this will allow us to scale up.

Please don't ask the question; "Why no SQL?". We are big fans of SQL and use it all the time in our day to day lives. We have however decided to go down the NoSQL route. If this one day doesn't live up to our scalability expectations we will be jumping straight back on the SQL train. FYI I am a big fan of MariaDB.
